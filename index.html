⚙️ 云端多端同步配置

💡 如何开启云同步？(只需1分钟)

  1. 访问 jsonbin.io，点击右上角用 GitHub 免费注册并登录。
  2. 新建一个数据箱 (Create Bin)，在代码框里输入 {"events":[]}，点击右上角的 Save。
  3. 保存后，网页地址栏最后的一串字符就是 Bin ID，复制填入上方。
  4. 点击网页左侧菜单的 API Keys，复制那串极长的 Master Key，填入上方的 API Key。
  5. 点击底部保存。之后你在手机或电脑上填入相同的两串代码，排期就会无缝实时同步！

🔐 数据直连你自己的专属数据库，绝对隐私安全。
